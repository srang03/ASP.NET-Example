[1] Calendar 컨트롤
	웹 페이지에 달력을 표현하는 컨트롤

[2] Adrotator
	웹 페이지에 광고 기능을 구현하는 컨트롤러로 특정 빈도수에 맞게 배너 광고를 띄울 수 있다.
	최근에는 js 라이브러리가 광고 출력 기능을 제공해서 애드로테이터 컨트롤의 사용 빈도 감소했다.
	XML 형태로 작성 후 불러올 수 있다.

[3] FileUpload 컨트롤
	로컬 PC에 있는 파일을 웹 사이트에 업로드 한다.
	if(Upload_File.HasFile)
	{
		Upload_File.SaveAs(Server.MapPath(".") + "\\files\\" + Upload_File.FileName);
		lbl_result.Text = "./files/" + Server.UrlEncode(Upload_file.Filename); 
	}
	
	자료실, 앨범, 음악 게시판에 필요한 파일 업롣 ㅡ컨트롤
	클라우드 서비스 BLOB 스토리지에도 저장이 가능하다.

[4] Wizard 컨트롤
	프로그램을 설치하는 절차처럼 순서가 있는 작업을 진행할 때 손 쉽게 구현할 수 있는 컨트롤

[5] XML 컨트롤
	XML 문서를 웹 페이지에 표현하기 위한 주요 기능 제공
	XML 파일에 대하여 디자인 후 XML 컨트롤을 사용하여 웹 페이지에 출력한다.
	<?xml> 오류 발생>
	1) .xml 파일로 데이터 로드
	2) .xsl 파일로 탬플릿 생성
	3) .aspx 파일에 <XML> 태그로 데이터 출력


[6] MultiView 컨트롤
	멀티뷰 컨트롤은 뷰 컨트롤과 일반적으로 같이 사용한다.
	멀티 뷰 컨트롤은 여러 모양으로 정의된 뷰 컨트롤 중에서 하나를 출력하는 컨테이너 역할을 한다.
	두 컨트롤을 이용하여 지정된 공간에 여러가지 내용을 동적으로 출력할 수 있다.

[7] View 컨트롤
	뷰 컨트롤은 각각의 모양을 정의하고, 멀티뷰와 함께 사용된다.

[8] Panel 컨트롤
	HTML <div> 태그와 같이 HTML 및 웹 폼의 구성 요소를 담는 컨테이너 역할을 수행한다.

[9] PlaceHolder 컨트롤
	틀(플레이스홀더) 컨트롤은 패널 컨트롤과 비슷한 역할 또는 특정 태그로 묶이지 않는 것이 특징이다.
	웹 브라우저로 표현되는 어떤 객체를 해당 객체만 깔끔하게 출력하려면 패널 컨트롤보다는 플레이스 홀더 컨트롤이 더 적합하다.
	Controls.Add() 메서드를 사용하여 컨테이너 안에 코드를 추가할 수 있다.

[10] Substitution 컨트롤
	캐싱 (성능 향상) 기능과 연관된 컨트롤이다.
	전체 페이지가 캐싱되는 상황에서 특정 문자열 부분은 캐싱되지 않는 최신 자료를 출력하고자 할 때 이 컨트롤을 사용한다.

	* 캐싱 : 웹 페이지를 매번 실행하는 대신 서버의 메모리에 지정된 시간 동안 내용을 담아 놓고 메모리에 담긴 내용을 출력하는 기능이다.
	페이지, 컨트롤 단위로 캐싱되는 기능이다.
