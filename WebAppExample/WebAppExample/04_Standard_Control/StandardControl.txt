[1] HTML 태그를 대체하는 표준 컨트롤
	ASP.NET 웹 폼은 순수 HTML5 태그로 UI 요소를 표현할 수 있다.
	이러한 UI 요소에 서버 측 기능을 넣고 싶다면, HTML 태그를 대체하는 표준 컨트롤을 사용할 수 있다.

	ASP.NET에서 제공하는 표준 컨트롤 중에는 하이퍼링크, 이미지, 이미지맵 ,테이블 등 기존에 HTML 태그로 정적으로 표현하던 내용을 
	서버 컨트롤을 사용하여 선언적 및 동적으로 HTML 태그를 생성하는 컨트롤을 제공한다.

[2] HyperLink 컨트롤
	일반적인 링크를 만드는데 사용한다.
	<a> 태그 렌더링

[3] Image 컨트롤
	<img> 태그 렌더링

[4] ImageMap 컨트롤

[5] Table 컨트롤

[6] BulletedList 컨트롤
	HTML의 목록, 즉 글머리 기호에 해당하는 리스트 형식의 텍스트를 표현할 때 사용한다.
	<ol> <ul> <li> 목록 등을 표현하는 데 사용한다.

[7] DropDownList 컨트롤
	HTML의 <select><option> 태그를 조합하여 ASP.NET 서버 컨트롤로 구현한 컨트롤이다.

[8] ListBox 컨트롤
	리스트 박스와 드롭 다운 리스트의 기능은 동일하지만 
	드롭 다운 리스트를 펼처 놓은 모양을 가지고 있고 멀티 선택이 가능하다.

[9] CheckBox 컨트롤
	<input type="CheckBox">와 같은 체크박스를 만든다.

[10] RadioButton 컨트롤
	라디오 버튼은 특성상 웹 페이지에서 그룹 단위로 다 하나의 항목만을 선택할 때 사용하는 컨트롤이다.

[11] HiddenField 컨트롤
	어떤 값을 웹 폼에 보관하여 사용자 눈에 보이지 않도록 하는 것이 목표이다.

[12] Literal 컨트롤
	리터럴 컨트롤은 웹 브라우저에 텍스트를 표현하는 방법 중 태그를 사용하지 않고 깔끔하게 텍스트를 출력하는 방법이다.
	따로 HTML 태그를 만들지 않고 입력한 데이터만 출력하는 컨트롤이다.
